# 健康管理系统前端页面

## 概述

健康管理系统前端基于Vue3 + TypeScript + Element Plus开发，包含9个核心业务模块，提供完整的老人健康监护管理功能。

## 功能模块

### 1. 数据大屏 (`dashboard/index.vue`)
- **统计卡片**: 设备总数、在线设备、老人总数、今日告警
- **设备在线率**: 环形图展示在线率统计
- **告警趋势**: 7天告警趋势折线图
- **告警分布**: 告警类型分布饼图
- **最新告警**: 实时告警列表
- **设备分布**: 地图展示设备位置
- **自动刷新**: 30秒自动刷新机制

### 2. 设备类型管理 (`device/type/index.vue`)
- **树形表格**: 支持父子级设备类型展示
- **分类管理**: 监测设备、定位设备、报警设备、其他设备
- **CRUD操作**: 新增、修改、删除、查询
- **展开折叠**: 树形结构展开折叠
- **权限控制**: 按钮级权限控制

### 3. 设备管理 (`device/index.vue`)
- **设备列表**: 分页展示设备信息
- **绑定管理**: 设备与老人绑定/解绑
- **状态显示**: 在线/离线状态、电量显示
- **导入导出**: Excel批量导入导出
- **高级筛选**: 多维度筛选条件

### 4. 告警管理 (`alert/index.vue`)
- **告警列表**: 级别颜色标识
- **告警详情**: 详细信息弹窗
- **处理功能**: 告警处理和忽略
- **地图显示**: 告警位置地图展示
- **导出功能**: 告警数据导出

### 5. 组织管理 (`organization/index.vue`)
- **树形表格**: 组织层级结构展示
- **类型管理**: 养老院、社区服务中心、医疗机构等
- **CRUD操作**: 完整的增删改查
- **联系信息**: 负责人、电话、邮箱、地址

### 6. 老人管理 (`elder/index.vue`)
- **老人档案**: 完整的健康档案管理
- **BMI计算**: 身高体重自动计算BMI
- **联系人管理**: 紧急联系人增删改查
- **护理等级**: 自理、半自理、不能自理、特护
- **风险等级**: 低风险、中风险、高风险
- **导入导出**: Excel批量操作

### 7. 体征数据 (`vitalSigns/index.vue`)
- **体征列表**: 异常值红色标识
- **异常检测**: 自动判断体征异常
- **趋势分析**: 体征数据趋势图表
- **数据录入**: 手动录入体征数据
- **导出功能**: 体征数据导出

### 8. 服务人员 (`serviceStaff/index.vue`)
- **人员管理**: 服务人员信息管理
- **类型分类**: 护理员、医生、康复师等
- **专业特长**: 专业技能描述
- **工作时间**: 工作时间安排
- **组织关联**: 所属组织管理

## 扩展说明

### 1. 图表集成
系统预留了图表集成接口，可以集成以下图表库：
- ECharts: 用于统计图表
- AntV G2: 用于数据可视化
- Chart.js: 轻量级图表库

### 2. 地图集成
系统预留了地图集成接口，可以集成以下地图服务：
- 高德地图: 国内主流地图服务
- 百度地图: 提供丰富的地图功能
- 腾讯地图: 企业级地图服务

### 3. 功能扩展
- 消息推送: WebSocket实时消息
- 文件上传: 支持多种文件格式
- 数据导入: Excel、CSV等格式
- 报表生成: PDF、Word报表导出

## 部署说明

### 1. 开发环境
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 2. 生产环境
```bash
# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 3. 环境配置
- 开发环境: `.env.development`
- 生产环境: `.env.production`
- API地址配置: `VITE_APP_BASE_API`
